Overview

This guide helps users understand how to navigate the CryptoPill client interface, interpret the main screens, and perform common actions fluidly.

Main Screens
1. Dashboard

Your entry point. It centralizes your mood tier, recent matrix changes, and quick alerts.

Mood Tier Card: Shows current tier (e.g., Neutral, Elevated, Distressed) calculated from mea‑mood.

24h Matrix Snapshot: Displays pct24h evolution for your enabled symbols.

Active Flags: Any risk or flow alerts appear here.

2. Assets View

Your symbol‑level breakdown.

Holdings Table: Amount, value, pct change.

Matrix Indicators: pct24h, rolling benchmarks, weights.

Vector Windows: Quick access to Str‑Aux vector summaries.

3. Sessions / Ops

Operational sessions that log every interaction with exchanges or internal ledger.

Session Timeline: Open/close stamps, binance pulls, ingestion results.

Flow Gaps: Shows if any symbol had missing windows.

4. Ledger / Cin‑Aux

Tracks imprint/luggage and PnL.

Imprint Table: All primary entries.

Luggage Table: Propagated effects.

PnL Reasoning: Shows realized/unrealized breakdown.

5. Settings

Define the universe of symbols and behaviors.

CSR/SSR: Universe selection.

Enable/Disable Symbols: Directly reflected downstream.

Benchmarks & Windows: User presets.

Typical User Flows
Check My Mood Tier

Enter dashboard.

Mood card displays current tier and deltas.

Tap for detailed moods: symbol contribution, thresholds, matrices used.

Inspect Symbol Behavior

Go to Assets.

Select symbol.

Inspect pct24h, windows, and vector lineage.

Review Ledger Movements

Go to Ledger.

Check imprint entries.

View their luggage propagation.

Read PnL explanations.

Adjust My Universe

Go to Settings.

Enable/disable symbols.

Refresh to see matrix updates.
